# SQLP 채점 기준

## 점수 배분

| 구분 | 문항 수 | 문항당 배점 | 소계 |
|------|---------|------------|------|
| 객관식 (전 과목) | 70문항 | 1점 | 70점 |
| 서술형 (전 과목) | 2문항 | 15점 | 30점 |
| **합계** | **72문항** | | **100점** |

### 과목별 객관식 문항 배분

| 과목 | 객관식 문항 수 |
|------|--------------|
| 3과목: SQL 고급활용 및 튜닝 | 40문항 |
| 1과목 + 2과목 합계 | 30문항 |
| **전체** | **70문항** |

> ℹ️ 1과목, 2과목의 세부 문항 수는 시험 회차에 따라 다를 수 있습니다.

## 합격 기준

```
✅ 합격 조건:
  1. 전 과목 평균 75점 이상
  2. 각 과목 40점 이상 (과락 없어야 함)
```

> ⚠️ 합격 기준은 회차별로 변경될 수 있으니 반드시 공식 사이트에서 확인하세요.

## 객관식 채점

- 정답 1개 선택
- 오답 시 0점 (감점 없음)
- 문항당 1점

## 서술형 채점

- 전 과목 통합 2문항, 문항당 15점
- 부분 점수 있음
- 핵심 키워드 포함 여부 중요
- 논리적 흐름과 정확성 평가

## 서술형 예시 유형

1. **SQL 실행 계획 분석**: 주어진 실행 계획을 보고 문제점 파악 및 개선 방안 제시
2. **인덱스 설계**: 특정 쿼리에 최적화된 인덱스 설계
3. **SQL 튜닝**: 비효율적인 SQL을 튜닝하여 개선된 SQL 작성
4. **데이터베이스 구조 분석**: Lock, 트랜잭션 관련 문제 분석

## 공부 전략

| 과목 | 비중 | 전략 |
|------|------|------|
| 1과목 | ~10% | 개념 정확히 이해, 용어 암기 |
| 2과목 | ~20% | SQL 실습 병행, 함수/조인 완벽 이해 |
| 3과목 객관식 | 40% | 인덱스·옵티마이저·튜닝 원리 이해 |
| 서술형 | 30% | 실행계획 분석 + SQL 튜닝 작성 연습 |

> 💡 서술형 2문항이 30점으로 합격에 큰 영향을 미칩니다. 실제 SQL 작성 연습을 많이 하세요.
