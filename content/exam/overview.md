# SQLP 시험 개요

## 자격 정보

| 항목 | 내용 |
|------|------|
| 자격명 | SQL 전문가 (SQL Professional, SQLP) |
| 주관기관 | 한국데이터산업진흥원 (K-DATA) |
| 자격 종류 | 국가공인 민간자격 |
| 응시 자격 | SQLD 취득 후 실무 경력 2년 이상, 또는 실무 경력 3년 이상 |

## 시험 과목

| 과목 | 주요 내용 | 문항 유형 |
|------|-----------|-----------|
| 1과목 | 데이터 모델링의 이해 | 객관식 |
| 2과목 | SQL 기본 및 활용 | 객관식 |
| 3과목 | SQL 고급활용 및 튜닝 | 객관식 + 단답형/서술형 |

## 합격 기준

- 전 과목 **총점 75점 이상** (100점 만점 기준)
- 각 과목 **40점 이상** 필수 (과락 기준)

> ⚠️ 합격 기준은 회차별로 변경될 수 있으니 반드시 공식 사이트에서 확인하세요.

## 시험 일정

시험은 연 2~3회 시행됩니다. 정확한 일정은 [K-DATA 공식 홈페이지](https://www.kdata.or.kr)에서 확인하세요.

## 시험 구성

```
총 시험 시간: 180분
- 객관식: 40문항
- 단답형/서술형: 10문항 (3과목)
```

## 준비 자료

- 공식 수험서: 「SQL 전문가 가이드」(한국데이터산업진흥원 발행)
- 기출문제: K-DATA 홈페이지 또는 수험생 커뮤니티 참고
